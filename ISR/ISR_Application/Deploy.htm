About BSS

Business Support Systems (BSS) are the set of components that a service provider users to run its business operations.Basically the purpose of BSS in Multinet is to centralize the order management, service provisioning and the complaint management, as currently we have multiple databases for different services and it would be difficult to manage the information because of multiple sources of customer's basic, billing and technical details which eventually lack tracking mechanism and adversely effects customer support.

Prerequisites

The following prerequisites are required:

  • Windows Installer 3.1
  • SAP Crystal Reports Runtime Engine for .NET Framework
  • Microsoft .NET Framework 4 (x86 and x64)

If these components are already installed, you can launch the application now. Otherwise, click the button below to install the prerequisites and run the application.
